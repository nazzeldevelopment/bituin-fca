## Authentication
- Login: `https://www.facebook.com/login.php`
- Logout: `https://www.facebook.com/logout.php`

## Messaging

- Upload: `https://upload.facebook.com/ajax/mercury/upload.php`

## GraphQL
- Endpoint: `https://www.facebook.com/api/graphql/`
